ZRI.info Roadmap

Post-1.0 plans to enhance style, admin tools, user experience, games, community, and retro-futuristic features.

🎛️ v1.1 – Core Interactions & Style Polish

  • 🖱️ Click Sounds & Hover FX: Subtle retro audio and visual feedback to make the site feel tactile.
  • 🧃 UI Juice: Animations and micro-interactions that make everything feel alive — buttons squish, text pulses, etc.
  • 🌀 Dynamic Backgrounds: Themes that react to time of day or user activity (stars move, waves shift, etc.).
  • 🧊 Cooldown Mode: Optional “chill” mode that reduces motion/sound for late-night or low-energy browsing.
  • 📼 Theme Presets: Switch between vaporwave, space noir, CRT green, or neon grid styles.

🛠️ v1.2 – Admin Power-Ups

  • 🗃️ User Roles: Add roles (admin, mod, guest) to users in local storage.
  • 🚫 Auto Ban Triggers: Ban based on suspicious behavior or too many attempts.
  • 📌 Persistent Bans: Ban by IP/device fingerprint, not just UID.
  • 🛎️ Activity Alerts: Notify on new users or logins via webhook.
  • 📈 Analytics Dashboard: Real-time user and game analytics for admins.
  • 🗳️ Moderation Queue: Approve or reject user submissions in a centralized queue.

📲 v1.3 – App Mode & UX

  • 📱 PWA Integration: Make zri.info installable as a web app.
  • 🌊 Animated Launch Splash: Stylized startup screen for mobile launch.
  • 🧭 Floating App Bar: Always-accessible navbar for games, blog, devlog.
  • ✨ Audio/Theme Toggles: Let users switch site theme + ambient sounds.
  • 🌐 Offline Mode: Cache key pages and games for offline access.
  • 👆 Gesture Controls: Swipe and tap gestures for intuitive navigation.

🎮 v1.4 – Games + Meta

  • 🎲 Dynamic Game List: Admin panel to add/edit games via JSON.
  • 🔄 Game Versioning: Add changelogs or update notes for games.
  • 🔍 Search + Tags: Let users search/filter by genre or keywords.
  • 🧾 Game Descriptions: Display blurbs and author info on click.
  • 🏆 Leaderboards: Per-game rankings for top players.
  • 🎖️ Cross-Game Achievements: Unlockable badges across multiple games.

🤝 v1.6 – Social & Collab

  • 🗨️ Game Comments: Lightweight per-game comment threads.
  • ⭐ Public Profiles: Users can create & share game lists.
  • 🎁 Dev Submissions: External devs can pitch games for listing.
  • 📡 API Endpoints: Embed or query game metadata externally.
  • 📺 Live Game Streaming: Users can stream gameplay to others.
  • 🎉 Community Events: Host game tournaments or dev challenges.

🌌 v1.7 – Retro-Future Horizons

  • 🎨 Pixel Art Editor: Create and share pixel art assets in a 90s sprite editor.
  • 🕹️ Virtual Arcade Hub: 3D-rendered arcade to browse games and socialize.
  • 🥚 Dynamic Easter Eggs: Randomized hidden interactions (e.g., secret links).
  • 🧩 Modular Plugins: Community-built widgets for custom UI and features.
